# Gamma

A C# console application featuring colorful ASCII art menu design with .NET 10.0 support.

## Features

- 🎨 **Colorful Console UI** - Enhanced terminal output with color support using Colorful.Console library
- 🎮 **Interactive Menu** - ASCII art-based menu system for user interaction
- ⚙️ **Modern .NET** - Built with .NET 10.0 for latest performance and features
- 🚀 **Cross-platform** - Runs on Windows, Linux, and macOS

## Requirements

- .NET SDK 6.0 or higher
- Git (for installation)

## Installation

### Quick Install (Linux/macOS)

```bash
wget https://raw.githubusercontent.com/lofi1130/gamma/main/gamma_install.sh
sudo bash gamma_install.sh
```

### Manual Build

1. Clone the repository:
```bash
git clone https://github.com/lofi1130/gamma.git
cd gamma/Gamma
```

2. Restore dependencies:
```bash
dotnet restore
```

3. Build the project:
```bash
dotnet build
```

4. Run the application:
```bash
dotnet run
```
